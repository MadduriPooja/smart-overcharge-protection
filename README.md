# Smart Overcharge Protection System

This project is a demo website that prevents mobile and laptop batteries 
from overcharging by monitoring charging time and giving alerts.

## Features
- Login page with name input
- Countdown timer to track charging duration
- Battery progress bar with color changes:
  - Green (safe)
  - Yellow (warning)
  - Red (critical)
- Alert system: bell sound + popup when charging exceeds limit
- Responsive design for mobile and desktop

## How to Run
1. Download or clone this repository.
2. Open `index.html` in your browser.
