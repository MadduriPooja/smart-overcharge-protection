# Smart Overcharge Protection System 🔋

A web-based application designed to **prevent battery overcharging** by monitoring charging cycles, setting reminders, and alerting users when the battery is fully charged.  
Built with **HTML, CSS, and JavaScript**, this project demonstrates responsive design, interactive dashboards, and practical problem-solving for everyday device usage.

---

## 🚀 Features
- **Home Page**: Clean landing section introducing the system.
- **Services Section**: Highlights monitoring, safeguard, and longevity features.
- **About Section**: Explains the purpose and importance of battery protection.
- **Demo Section**: Interactive timer and alarm sound to simulate charging alerts.
- **Contact Section**: Simple form for user interaction.

---

## 🛠 Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
- **Deployment**: GitHub Pages
- **Assets**: Custom screenshots used for visuals

---

## 📱 Responsive Design
- Desktop: Full navigation bar with links (`Home, Services, About, Demo, Contact`)
- Mobile: Hamburger menu toggle for easy navigation
